# dungeon-wizard

A roguelike written in coffescript using rot-js. Mostly as a means to
learn coffescript :)

## Play

A build is deployed at http://space-wizards.net/games/dungeon-wizard/ if
you would like to try it out.

## Building

```
npm install
gulp build
```
